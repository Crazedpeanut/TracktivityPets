How Tracktivity Pets Works
===========================

Installation
-----------------------------

Dependencies
~~~~~~~~~~~~~~

* Ubuntu 14.04 LTS (May work on other operating systems however)
* Python3
* pip3
* Nginx (We used version 1.9)
* mySQL or mariaDB server 
* Python libraries
* django
* fitbit
* django-fitbit
* django-celery
* mysqlclient
* gunicorn
* Virtualenv
* RabbitMQ
* Celery
